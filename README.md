# msf-modules

Repository for my Metasploit Framework modules

# Installation

`git clone https://github.com/japkettu/msf-modules`

Inside msfconsole

`loadpath <path_to_msf-modules_folder>/modules`

Example

`loadpath ~/github/msf-modules/modules`
